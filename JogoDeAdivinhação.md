Jogo de Adivinhação – Java

Um jogo simples de adivinhação criado em Java, onde o programa gera um número aleatório entre 0 e 100 e o usuário tem 5 tentativas para acertar.

---

Funcionalidades
- Gera um número aleatório.
- Permite 5 tentativas ao usuário.
- Informa se o palpite é maior ou menor que o número real.
- Exibe o resultado final (acertou ou perdeu).

---

Tecnologias utilizadas
- Java
- Scanner
- Random
